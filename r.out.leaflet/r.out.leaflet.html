DESCRIPTION

r.out.png.leaflet Maps are prepered to be overlays on some map layer such as OpenStreetMap. Outputs:
  • PNG images in Pseudo-Mercator (aka Spherical Mercator, Mercator, Web Merkator or Google Mercator) projection (EPSG:3857)
  • JS file with JavaScript list variable initialized by object literals contaning all data about the map and images
  • CSV file contaning the same data as JS file (not completed yet)

NOTES

This module uses r.out.png.proj to export PNG images in the Spherical Mercator projection (EPSG:3857).

Image geographic bounds are extracted from .wgs84 file produced by r.out.png.proj module.
